  return-to-cultivation. Each step includes the exact field values to enter.

  - Site: replica1.local running ERPNext v15 + pluviago custom app
  - Company: "Pluviago Biotech Pvt. Ltd." (abbreviation: PB)
  - Login as Administrator or a user with Pluviago Admin role
  - The following ERPNext Suppliers must exist (create if not):
      • SRL Chemicals
      • Qualigens Fine Chemicals
      • HIMEDIA Laboratories
      • Molychem
  - The following Item records must exist (item_group = "Raw Materials"):
      • CHEM-001  Calcium Chloride Dihydrate
      • CHEM-002  Magnesium Sulphate Heptahydrate
      • CHEM-003  Sodium Chloride
      • CHEM-004  Manganese II Chloride Tetrahydrate

      • CHEM-011  Ferric Citrate
      • CHEM-012  Sodium Nitrate
      • CHEM-013  Potassium Dihydrogen Phosphate
      • CHEM-014  Potassium Phosphate Dibasic
  - Warehouse: "Chemical Store RT - PB" (leaf warehouse under Pluviago)
  - UOM: "g", "mg", "mL", "L" must exist in UOM master
